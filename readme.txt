WP Responsive Portfolio Gallery is an enhanced tool to create Responsive Portfolio Gallery  for WordPress Websites.
== Description ==
WP Responsive Portfolio Gallery is perfect plugin  for creating various portfolios or gallery within various views. The product allows adding descriptions and titles for each portfolio gallery. It is fast and responsive with a easy user interface and totally AJAX based.There is no need to wait for pages to reload.

== Installation ==
1. Go to plugins in your dashboard and select 'add new'
2. Search for 'WP Responsive Portfolio Gallery' and install it
3. Go to Dashboard > Responsive Portfolio Gallery and add items for porfolio
4. Use this shortcode [responsive-portfolio] in any page
5. Now visit that page and see plugin in action
